# Tic Tac Toe

A simple implementation of the classic Tic Tac Toe game written in Rust.

## Features

- Two-player gameplay.
- Command-line interface.
- Validates moves and checks for a winner or draw.

## Prerequisites

- Rust installed on your system. You can install it from [rust-lang.org](https://www.rust-lang.org/).

## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/tic_tac_toe.git
    cd tic_tac_toe
    ```

2. Build the project:
    ```bash
    cargo build
    ```

3. Run the game:
    ```bash
    cargo run
    ```

## How to Play

- Players take turns to input their moves by specifying the row and column.
- The game ends when one player gets three in a row (horizontally, vertically, or diagonally) or when the board is full.
